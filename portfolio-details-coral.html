Capturing Coral Tentacles

Project information

  • Category: Desktop Application
  • Tech Stack: Python, SQL, CSS, YOLOv5, Roboflow
  • Client: Bergen County Academies Coral Research Program
  • Project date: September 2022 - June 2023
  • Project URL (Main Application): https://github.com/aparna-pillai/Capturing-Coral-Tentacles-Project
  • Project URL (Admin Application): https://github.com/aparna-pillai/Admin-App-Capturing-Coral-Tentacles

o Expedited data collection time using Roboflow and YOLOv5 in a desktop app for precise coral tentacle detection and counting

o Enabled efficient image and user data storage and access with a built-in record system, with the function to export the data to Excel

o Created an admin application for user account maintenance, streamlining adding/deleting users, and password management
